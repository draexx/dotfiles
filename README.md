# Configuración
Archivos de configuración de mi sistema operativo.

## Requerimientos
Para utilizar estas configuraciones, asegúrate de tener instalados los siguientes programas:
- `zsh`
- `oh-my-zsh`
- `starship`
- `VSCode` o `Cursor`
- `glances`
- `eza`
- `duf`
- `fastfetch`
